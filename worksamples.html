Conversion - C++

This was our 4th and final assignment in our C++ class. This reads in a C++ file via user input, and converts it into an HTML file. There were some rules we had to follow. The user had to input the file they wish to convert (in this case, the CPP file), and had to name what the output file was going to be (ex: CPPTwo). As the file was being read, if there was a less than symbol (<), or a greater than symbol (>), we had to change it to "& lt;" and "& gt;" respectfully. In addition, there needed to be a "PRE" tag at the beginning and end of the file respectfully.

I remember having a rough time getting started with the assignment as I was overthinking it way too much. Once I figured out where to start, and broke things down into smaller tasks, it was smooth sailings from there. Overall, I was pleased with the result, and demonstrates how far I've come in programming over the 2 years I spent at the NSCC.
